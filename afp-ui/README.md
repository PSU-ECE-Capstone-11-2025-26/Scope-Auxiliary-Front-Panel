# AFP-UI
This is the touch screen user interface for the Tektronix Auxiliary Front Panel.

## Development
AFP-UI is developed using Godot Engine, currently: `Godot Engine 4.6.1 .NET`

1. [Install a compatible .NET SDK][dotnet-download] (>=8.0)
2. [Download Godot][godot-download]. Godot is a portable application. You can just unzip it, and run it!
3. Clone this repo
4. Open the Godot project (afp-ui) in Godot.
5. Off you go!

Why a game engine? Why Godot? Godot is free and open source, and is
self-contained as a small, portable binary for many OSes and architectures.
Godot has an extensive UI system with more than enough features for non-game
applications. All this makes Godot a competitive option compared with Qt, GTK,
 or Electron. See the [Godot Docs][godot-apps].
