# AFP Communication Software

This directory contains the source code for the main communication software, handling:
- UART communication between the main processor (Raspberry Pi) and the interface controls microcontroller (Raspbery Pi Pico).
- Decoding messages from the microcontroller into corresponding SCPI commands
- Sending SCPI commands to the scope
- Managing and storing oscilloscope state
- Managing and storing AFP configuration
